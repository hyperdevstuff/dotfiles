# Dotfiles (Always WIP)

## Applications

- Compositor -> [Hyprland](https://hypr.land)
- Terminal -> [Kitty](https://sw.kovidgoyal.net/kitty/)
- Bar -> [Waybar](https://github.com/Alexays/Waybar)
- Notification daemon [swaync](https://github.com/ErikReider/SwayNotificationCenter)
- App-launcher [rofi-wayland](https://github.com/in0ni/rofi-wayland)
- Code-editor [neovim](https://neovim.io)
- File explorer -> [yazi](https://yazi-rs.github.io/), [dolphin](https://apps.kde.org/dolphin/)
- shell - [zsh](https://www.zsh.org/) with [starship](https://starship.rs/)
